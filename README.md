# Rise Browser

Современный браузер на базе PySide6 (Qt6) с поддержкой I2P, VPN, AI‑чата и широкими возможностями кастомизации.

[![Download](https://img.shields.io/badge/Download-Latest-blue)](https://github.com/CkFw/Rise/releases/latest)
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

## Возможности

- 🔹 **Многозадачность** – вкладки, история, закладки, восстановление сессии
- 🔹 **I2P‑режим** – работа с .i2p сайтами через встроенный i2pd (требуется отдельная установка i2pd)
- 🔹 **VPN & прокси** – поддержка SOCKS5, HTTP, Shadowsocks, Trojan, VMess, VLESS, Tuic, Hysteria, Juicity
- 🔹 **Обход DPI** – встроенная поддержка утилит GoodbyeDPI / zapret
- 🔹 **Блокировщик рекламы** – три режима: простой (списки доменов), расширенный (EasyList), интеллектуальный
- 🔹 **AI‑чат** – бесплатный GPT-чат с возможностью озвучивания ответов (Edge‑TTS)
- 🔹 **Управление паролями** – импорт/экспорт в CSV
- 🔹 **Кастомизация интерфейса** – верхняя панель, адресная строка, фон домашней страницы, цвета вкладок, иконки, автоскрытие
- 🔹 **Умный буфер** – предзагрузка популярных сайтов для быстрого доступа
- 🔹 **Энергосбережение** – выгрузка неактивных вкладок, удаление скрытых видео из DOM
